# `zeeman` – Scripts for Zeeman effect analysis and related astrophysical studies

This folder contains Python scripts developed for the mathematical and phenomenological study of the Zeeman effect in the context of the pentadic theory, as well as for analyzing astrophysical data (magnetars, pulsars) and exploring higher-dimensional structures (Bott periodicity, Nebe lattice). All scripts are designed to be run independently and produce figures and reports.

---

## Prerequisites

- Python ≥ 3.8
- Required packages: see the main [`requirements.txt`](../requirements.txt) or [`environment.yml`](../environment.yml) at the root of the repository.
- Some scripts (especially Fermi‑LAT analysis) require `fermipy` (available via conda) and an internet connection to download data.

---

## Scripts overview

### 1. `Étude_mathématique_et_analogies_physiques_du_réseau_de_Nebe_en_dimension_72.py`
**Mathematical study of the Nebe lattice (dimension 72) and physical analogies**

This script implements the fundamental properties of the Nebe lattice (arXiv:1008.2862): dimension, minimal norm, kissing number, automorphism group. It also explores numerical coincidences with the fine structure constant and the resonance magnetic field \(B_0 = 8\,\mathrm{T}\).  
**Output:** console summary, figure `preuve_dimension_72_math.png`.

**Usage:**  
```bash
python "Étude_mathématique_et_analogies_physiques_du_réseau_de_Nebe_en_dimension_72.py"
```

---

### 2. `FERMI-LAT_MAGNETAR_ANALYSIS_v2.0.py`
**Fermi‑LAT data analysis for magnetars, searching for the 0.2 GeV resonance (octave 6)**

This script reproduces and extends the analysis of Ramakrishnan & Desai (2024). It uses `easyFermi` to download public Fermi‑LAT data for a sample of magnetar flares, performs a maximum‑likelihood analysis with `fermipy`, and looks for an excess at 0.2 GeV (the predicted signature of the pentadic model).  
**Output:** summary figure `magnetar_analysis_summary.png`, detailed text report `fermi_magnetar_report.txt`.

**Note:** Requires a conda environment with `fermipy` installed. Execution may take several hours and requires a few GB of disk space.

**Usage:**  
```bash
python FERMI-LAT_MAGNETAR_ANALYSIS_v2.0.py
```

---

### 3. `BOTT_PERIODICITY_v2.0.py`
**Bott periodicity and higher octaves of dimension 72**

Starting from the fundamental dimension 72, this script applies Bott periodicity (factor 8) to generate higher octaves: 72, 576, 4608, … It incorporates an empirical projection factor derived from observations to correct the pure theoretical scaling \(B \propto 1/D\). For each octave, it computes the predicted magnetic field, characteristic length, and compares with experimental domains (laboratory, white dwarfs, pulsars, magnetars, cosmology).  
**Output:** figure `bott_octaves_corrected.png`, text report `bott_octaves_report.txt`.

**Usage:**  
```bash
python BOTT_PERIODICITY_v2.0.py
```

---

### 4. `CROSS-VALIDATION_PENTADIQUE_v1.0.py`
**Cross‑validation of pentadic parameters across different experimental regimes**

This script simulates various experimental setups (laboratory, pulsed fields, optical/X‑ray pulsars, magnetars) and computes a figure of merit that combines the predicted effect size, signal‑to‑noise ratio, cost, duration, and technological readiness. It helps prioritize future observations.  
**Output:** figure `cross_validation.png`, console table.

**Usage:**  
```bash
python CROSS-VALIDATION_PENTADIQUE_v1.0.py
```

---

### 5. `ZEEMAN-PENTADIC-ANALYSIS_v1.0.py`
**Extraction of pentadic model parameters and predictions**

Using synthetic data simulating laboratory and pulsar Zeeman measurements, this script fits the parameters of the pentadic model: the fundamental g‑factor \(g_0\), lattice coupling \(\delta_\text{lattice}\) and resonance scale \(B_0\), and the bicosmic coupling \(\delta_\text{bicosmic}\). It then computes an effective dimension of the underlying lattice and generates quantitative predictions for future experiments (quantum beats, satellite lines in pulsar spectra, periodic modulation of g).  
**Output:** figure `pentadic_predictions.png`, detailed text report `pentadic_predictions_report.txt`.

**Usage:**  
```bash
python ZEEMAN-PENTADIC-ANALYSIS_v1.0.py
```

## Common notes

- All scripts save output figures and text reports in the current working directory (the folder from which they are launched). You may want to create a dedicated `output/` folder.
- Some scripts rely on the JSON file `pentades_72_finales.json` generated by the modules in the [`../pentadic`](../pentadic) folder. Make sure to run the relevant pentadic modules first if needed.
- For the Fermi‑LAT analysis, a stable internet connection is required to download the data. The script automatically caches downloaded files, so subsequent runs are faster.
- If you encounter any missing dependencies, please refer to the main `requirements.txt` and `environment.yml` files at the project root.
